## Actividad de Caso Estudio

**Descripción de la actividad o tarea**

-	Realizar un sistema de gestión para una librería, la cual constaba de uso de Base de datos (MySQL), JSP, JDBC y de base usando Java con Apache Maven.
  
**Desafío principal**
-	El mayor desafío fue comprender las bases de JSP, JDBC, su implementación y como se combina con el uso de Java.
  
**Solución Propuesta**
-	Mi principal solución en ese momento fue realizar de forma pausada cada parte del CRUD, identificando cada elemento y como se conecta cada método para hacer las solicitudes hacia la Base de Datos para obtener y enviar información.
  
**Principales aprendizajes alcanzados**
-	Optimización de mi tiempo para el estudio, gestión del uso de IA para mejorar el aprendizaje de los sistemas antes mencionados.
  
**Métricas de impacto logradas**
-	Cumplimiento de entrega del trabajo en los tiempos correspondientes.
  
**Habilidades Técnicas Aplicadas**
-	Administración de tiempo, uso de Inteligencia Artificial, gestión del proyecto para dividir en logros mas pequeños para ir completando.
  
**Justificación de porque se eligió este proyecto**
-	Principalmente porque las bases son el comienzo, es lo que marca la diferencia en la trayectoria de cualquier lenguaje y es lo que finalmente termina llamando la atención.
